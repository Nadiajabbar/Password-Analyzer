 ##🔐 AI Password Strength Analyzer

An AI-powered web application that analyzes the strength of passwords and predicts whether a password is **Weak, Medium, or Strong** using a Machine Learning model. The project is built with **Python, Flask, HTML, CSS, and Scikit-learn**.

---

## 📌 Features

- Analyze password strength instantly.
- Machine Learning-based prediction.
- User-friendly web interface.
- Fast and accurate password classification.
- Responsive design.

---
🛠 Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Machine Learning
- Scikit-learn
- Pickle

🚀 Installation

 1. Clone the repository

```bash
git clone https://github.com/Nadiajabbar/Password-Analyzer.git
```

 2. Go to the project folder

```bash
cd Password-Analyzer
```

### 3. Install the required packages

pip install flask scikit-learn pandas numpy

### 4. Run the application

python app.py


### 5. Open in Browser

http://127.0.0.1:5000
